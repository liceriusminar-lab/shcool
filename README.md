  🎮 Tetris
Онлайн тетрис с глобальной таблицей рекордов, написанный на C++ с использованием Qt Framework. Проект выполнен в рамках учебной программы колледжа.

   📖 О проекте
Это классическая игра Тетрис:
- Онлайн таблица рекордов — результаты видны всем игрокам
- Регистрация никнейма привязанного к устройству
- Ghost piece — тень показывающая куда упадёт фигура
- Hold — возможность отложить фигуру
- Анимация очистки линий
- Постепенное ускорение по уровням
- Градиентная графика

   🛠 Технологии
- **C++17** — основной язык
- **Qt5/Qt6** — графика, сеть
- **Python + Flask** — сервер рекордов
- **Railway** — хостинг сервера
- **GitHub Actions** — автоматическая сборка

   🚀 Запуск
  bash
git clone https://github.com/liceriusminar-lab/tetris.git
cd tetris
mkdir build && cd build
cmake ..
make -j$(nproc)
./Tetris

  🎮 Управление
| Клавиша | Действие |
|---------|----------|
| ← →     | Движение влево/вправо |
| ↑       | Поворот фигуры |
| ↓ / Пробел | Ускорить падение |
| C       | Hold — отложить фигуру |

  🏆 Таблица рекордов
Рекорды сохраняются на сервере и доступны всем игрокам в реальном времени. Никнейм привязывается к устройству — никто другой не сможет использовать твой ник.
